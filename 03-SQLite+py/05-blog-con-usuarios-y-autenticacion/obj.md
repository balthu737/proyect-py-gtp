5) ✍️ Blog con usuarios y autenticación

🎯 Objetivo pedagógico:

Manejo de usuarios y autenticación.

Introducción a full-text search (FTS5 en SQLite).

🏗️ Objetivo práctico:
Un blog con usuarios registrados, posts en Markdown y comentarios.

📌 Conceptos de Python:

Hashing de contraseñas (werkzeug.security).

Manejo de sesiones en Flask.

Render de Markdown (markdown2).

📌 Conceptos de SQL/SQLite:

Tablas con roles y permisos.

Full-text search (CREATE VIRTUAL TABLE ... USING fts5).

Restricciones de unicidad (UNIQUE).
